# 中文提示词生成说明

## 🎯 功能说明

系统已配置为**生成中文提示词**，支持：

1. ✅ **中文对话**：可以用中文描述你的创作想法
2. ✅ **中文生成**：系统会生成中文提示词
3. ✅ **中英文混合知识库**：知识库可以包含中英文提示词

## 📝 使用流程

### 1. 准备数据（如果知识库包含中文）

如果你的 Excel/CSV 文件中包含中文提示词，直接处理即可：

```bash
python process_data.py
```

系统会自动识别并保留中文内容。

### 2. 构建索引

```bash
python build_index.py
```

### 3. 使用系统

```bash
streamlit run app.py
```

在界面中输入中文想法，系统会生成中文提示词。

## 🔄 重新处理现有数据

如果你已经处理过数据，但想改为支持中文生成，需要：

1. **重新处理数据**（如果原始数据包含中文）：
   ```bash
   python process_data.py
   ```
   新的 ETL Pipeline 会保留中文内容。

2. **重新构建索引**：
   ```bash
   python build_index.py
   ```

## 💡 提示

- **知识库语言**：可以是纯英文、纯中文，或中英文混合
- **查询语言**：支持中文和英文查询
- **生成语言**：始终生成中文提示词
- **技术参数**：技术参数（如 "8k", "masterpiece"）可以保留英文，这是标准做法
